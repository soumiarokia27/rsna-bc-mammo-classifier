About the Project

MammoSense is a web application designed to classify mammogram images into breast density categories B and C using deep learning technology. This application was created as a gift and a tool to help demonstrate the potential of AI in breast cancer screening.

The underlying model is based on EfficientNetB3, a powerful deep learning architecture known for its efficiency and performance in image classification tasks. The model was trained on a dataset from the RSNA Breast Cancer Detection challenge, focusing specifically on distinguishing between scattered fibroglandular densities (category B) and heterogeneously dense breast tissue (category C).

This project combines the latest in artificial intelligence technology with an intuitive, user-friendly interface to make advanced mammogram analysis more accessible.

Understanding Breast Density

What is Breast Density?

Breast density refers to the amount of fibroglandular tissue in the breast compared to fat. Dense breast tissue appears white on a mammogram, while fatty tissue appears dark. Different levels of density are categorized into four groups by radiologists:

  • Category A: Almost entirely fatty
  • Category B: Scattered areas of fibroglandular density
  • Category C: Heterogeneously dense
  • Category D: Extremely dense

Why Breast Density Matters

Breast density is important for several reasons:

  • Higher breast density can make it harder to detect cancer on a mammogram
  • Dense breast tissue is a known risk factor for developing breast cancer
  • Women with dense breasts may benefit from additional screening methods
  • Many states have laws requiring that women be notified if they have dense breasts

Focus on Categories B and C

B

Scattered Fibroglandular Densities

This category indicates that the breast has scattered areas of density, but the majority of the breast tissue is non-dense or fatty. Mammograms of this type are generally easier to read and interpret.

C

Heterogeneously Dense

In this category, the breast has many areas of fibroglandular tissue density. This can make it harder to see small masses on a mammogram, as they might be hidden by the dense tissue.

Note: While MammoSense currently focuses on distinguishing between categories B and C, future versions may expand to include all four density categories.

The Technology Behind MammoSense

Deep Learning Model

MammoSense utilizes EfficientNetB3, a state-of-the-art convolutional neural network architecture known for its balance of accuracy and computational efficiency.

The model was trained on thousands of mammogram images to recognize the subtle patterns that distinguish different breast density categories.

Advanced Image Processing

Before analysis, mammogram images undergo sophisticated preprocessing techniques to enhance features and standardize inputs:

  • Contrast Limited Adaptive Histogram Equalization (CLAHE)
  • Non-local means denoising
  • Normalization and resizing

Modern Web Application

The front-end interface is built with modern web technologies to provide a seamless, intuitive user experience:

  • Flask for server-side processing
  • Tailwind CSS for responsive design
  • JavaScript for interactive features
  • Secure file handling and processing
